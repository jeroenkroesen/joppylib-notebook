# joppylib-notebook
A simple Jupyter interface to use [JopPyLib](https://github.com/jeroenkroesen/joppylib).  
  
  
For more information refer to [JopPyLib](https://github.com/jeroenkroesen/joppylib).  
  
  
## Installation (using uv)
Clone this repository:  
```sh
git clone https://github.com/jeroenkroesen/joppylib
```
  
Go into the repo:
```sh
cd joppylib
```  
  
Create virtual env and install dependencies:
```sh
uv sync
```
